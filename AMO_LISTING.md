## Description

**Streamline your Feedly reading workflow with batch article opening.**

### The Problem
You save articles in Feedly on your phone (via NetNewsWire, Reeder, or the Feedly app), but opening them one-by-one on desktop is tedious and time-consuming.

### The Solution
Feedly Saved Opener opens all your saved articles in browser tabs with a single click, automatically unsaving them as they open. Perfect for batch-processing your reading list.

---

## ✨ KEY FEATURES

**⚡ Batch Opening**
- Open 1-100 articles at once (customizable)
- Optimized 50ms delay between tabs
- Opens in background tabs (won't disrupt your work)

**🔄 Auto-Unsave**
- Automatically removes save/star as articles open
- No manual cleanup needed
- Keeps your reading list organized

**📊 Badge Notifications**
- Shows saved article count on toolbar icon
- Updates every 5 minutes
- Know at a glance how many articles are waiting

**⚙️ Customizable**
- Adjust batch size to your preference
- Settings persist across sessions
- One-time setup, works forever

**🔐 Privacy & Security**
- No analytics or tracking
- Token stored locally in encrypted storage
- Direct Feedly API communication only
- Open source and auditable

**🆓 Free Tier Compatible**
- Works with free Feedly accounts
- Uses Feedly's developer tokens (250 requests/day)
- Process ~125 articles daily

## 📖 HOW TO USE

**First-Time Setup (2 minutes):**
1. Install the extension
2. Click the toolbar icon
3. Click "Get Feedly Token"
4. Copy your token from Feedly
5. Paste and save
6. Done!

**Daily Use (10 seconds):**
1. Click the extension icon
2. See your saved article count
3. Adjust batch size if needed
4. Click "Open X Articles"
5. Articles open and unsave automatically

## 🔧 FEATURES IN DETAIL

**Smart Batch Processing**
Choose how many articles to open at once. Smaller batches (10-20) are less overwhelming. Larger batches (50-100) are faster for power users.

**Real-Time Updates**
The badge shows your current saved article count. Click to refresh anytime. Settings update instantly as you type.

**Keyboard Shortcuts**
Press 'R' in the popup to refresh the count. Enter/Space to open batch when focused.

Clear error messages if token expires or connection fails. Easy token reset process.

Beautiful gradient design with smooth animations. Clean, intuitive layout. Professional polish throughout.

## 🔒 PRIVACY POLICY

**Zero Data Collection**
- No analytics

## 🐛 TROUBLESHOOTING

**Common Issues:**

*"No saved articles showing"*
- Click "Refresh Count"
- Check you're using the correct Feedly account
- Verify articles are saved in Feedly web interface

*"Token doesn't work"*
- Generate a new token at feedly.com/i/console
- Make sure you copied the complete token
- Remove old token and add fresh one

*"Articles don't open"*
- Check popup blocker is disabled
- Try reducing batch size
- Ensure Firefox is up to date

**More Help:**
- Full documentation: GitHub README
- Report bugs: GitHub Issues
